# Self-assessment

During my internship at Amazon Web Services Vietnam from April 17, 2026 to July 10, 2026, I had the opportunity to learn, train, and apply the knowledge I accumulated at university to real-world work, while gradually building the necessary skills for my future career.

Specifically, I participated in building the **Cloud Nexus** project, a network security simulation and analysis platform that allows users to design network diagrams and use AI (Google Gemini) to detect vulnerabilities, simulate attack paths, and recommend defense measures. In this project, I was responsible for drawing the **Architecture Diagram** on Draw.io to describe the system workflow on AWS. It details every processing step: from user access to the frontend, authentication via Cognito, request submission through API Gateway, asynchronous processing via SQS → EventBridge Pipes → Step Functions → Lambda, to data storage in DynamoDB and S3, calling AI for result analysis, sending alerts via SNS, and monitoring logs via CloudWatch. Through this work, I gained a deeper understanding of how serverless components on AWS work together in a real system, and how to present data flow and processing flow in a visual, easy-to-understand manner.

Throughout my internship at the company, I always strived to complete assigned tasks well, comply with company regulations, and proactively exchange ideas with colleagues to improve work efficiency and quality.
