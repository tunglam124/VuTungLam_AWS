# Cloud Nexus — Nền tảng Mô phỏng & Phân tích Bảo mật Mạng trên AWS

### Tổng quan
Cloud Nexus là nền tảng mô phỏng và phân tích bảo mật mạng (Threat Modeling Platform) dành cho các chuyên gia an ninh mạng và kiến trúc sư hạ tầng. Hệ thống cho phép người dùng vẽ sơ đồ mạng (network topology), sau đó sử dụng AI (Google Gemini) để tự động phát hiện lỗ hổng, mô phỏng đường đi tấn công, và đề xuất biện pháp phòng thủ. Toàn bộ backend chạy serverless trên AWS (Lambda, API Gateway, DynamoDB, SQS, SNS, Step Functions) và triển khai qua AWS CDK.
