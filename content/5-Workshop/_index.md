# Cloud Nexus — Threat Modeling Platform on AWS

### Overall
Cloud Nexus is a Threat Modeling Platform for cybersecurity professionals and infrastructure architects. Users can design network topologies visually, then leverage AI (Google Gemini) to automatically detect vulnerabilities, simulate attack paths, and recommend defense measures. The entire backend runs serverless on AWS (Lambda, API Gateway, DynamoDB, SQS, SNS, Step Functions) and is deployed via AWS CDK.
